# Duke Program User Guide

## Input commands 
### Add to task
- todo [description]
- deadline [description]/[by]
- event [description]/[from]/[to]

### Manipulate tasks
- mark [task number]
- delete [task number]

### Get tasks
- list
- find [search]

### Exit program
- bye
