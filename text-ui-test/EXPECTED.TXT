Hello! I'm Duke.
To add general tasks to the todo list, simply type in the task and press enter.
You can specify the type of tasks by using 'todo' / 'deadline' / 'event' keyword before the task description.
You can keep track of deadlines by typing 'deadline' followed by task description + '/by' to specify the time of the deadline.
You can also add events to the list by typing 'event' followed by task description + '/from' + starting time of event + '/to' ending time of event.
Type 'list' to view the current todo list.
Type 'mark' and task event index to mark the task as done and type 'unmark' and index to undo the task.
